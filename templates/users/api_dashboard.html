API Quick Start

Our API gives you full programmatic access to the Igbo Archives. You can search historical data, retrieve books, and upload new cultural archives or folklore directly from your own applications.

Example Request 
curl -X GET {{ api_base }}archives/ \
  -H "Authorization: Token YOUR_KEY"
import requests

url = "{{ api_base }}archives/"
headers = {"Authorization": "Token YOUR_KEY"}

response = requests.get(url, headers=headers)
print(response.json())
const url = "{{ api_base }}archives/";
const options = {
  method: 'GET',
  headers: { 'Authorization': 'Token YOUR_KEY' }
};

fetch(url, options)
  .then(res => res.json())
  .then(json => console.log(json))
  .catch(err => console.error(err));

AI Agent Connectivity

Connect AI agents like Claude or Cursor directly to our archives. This lets your AI "see" and "use" cultural data as their own tools.

MCP Connection Endpoint (SSE)

READY

Use this URL in your MCP client's configuration.

{{ mcp_endpoint }}

Client Configuration

{
  "mcpServers": {
    "igbo-archives": {
      "serverUrl": "{{ mcp_endpoint }}",
      "headers": {
        "Authorization": "Token YOUR_KEY",
        "Content-Type": "application/json"
      }
    }
  }
}
{
  "mcpServers": {
    "igbo-archives": {
      "command": "curl",
      "args": [
        "-X", "POST",
        "-H", "Authorization: Token YOUR_KEY",
        "-H", "Content-Type: application/json",
        "{{ mcp_endpoint }}"
      ]
    }
  }
}
