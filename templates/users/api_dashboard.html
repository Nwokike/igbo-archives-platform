Model Context Protocol

Connect your favorite AI tools like Claude, Cursor, and IDEs directly to the Igbo cultural archives.

What is it?

MCP is an open standard that lets AI agents discover and use platform tools. Any MCP-compatible client can query archives, search books, and interact with cultural lore through structured commands.
