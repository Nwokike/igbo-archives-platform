Query Parameters

  • ?search=keyword — Full-text search across title, description, caption
  • ?type=image|video|audio|document — Filter archives by type
  • ?category=slug — Filter archives by category slug

MCP (Model Context Protocol)

What is MCP?

MCP is an open standard that lets AI agents (like Claude, Cursor, or custom bots) discover and use your platform's tools. Any MCP-compatible client can query archives, search books, and create lore posts through structured commands.

MCP Endpoint

{{ mcp_endpoint }}

Available Tools

archives — List, search, create, update, delete archives. Plus: featured, recent.
books — List, search, create, update books. Plus: top_rated, rate, ratings.
lore — List, search, create, update, delete lore posts.
categories — List and view archive categories.

Connect from Claude Desktop

Add this to your claude_desktop_config.json:

{
  "mcpServers": {
    "igbo-archives": {
      "url": "{{ mcp_endpoint }}",
      "headers": {
        "Authorization": "Token YOUR_API_TOKEN"
      }
    }
  }
}

Connect from Cursor / VS Code

Add to your .cursor/mcp.json or VS Code MCP settings:

{
  "servers": {
    "igbo-archives": {
      "type": "http",
      "url": "{{ mcp_endpoint }}",
      "headers": {
        "Authorization": "Token YOUR_API_TOKEN"
      }
    }
  }
}
